# 📈 Learning Philosophy

Most repositories focus only on using machine learning libraries.

This repository also focuses on understanding **how machine learning works under the hood** by implementing algorithms from scratch, visualizing concepts, experimenting with different approaches, and documenting findings throughout the learning process.

The objective is to move beyond simply using AI tools and develop a strong understanding of the mathematics, intuition, and engineering behind them.
